Frontend

React-projektin luominen npm create vite@latest nimi -- --template react
Node paketti cd nimi npm install
Projektin käynnistäminen npm run dev
JSON-server projektin juurihakemistoon tiedosto db.json
JSON-serverin alustus npx json-server --port=3001 --watch db.json
Axios-kirjaston asennus npm install axios
JSON Server projektin sovelluskehityksen aikaiseksi riippuvuudeksi npm install json-server --save-dev
tiedoston package.json osaan scripts lisäys "server": "json-server -p3001 --watch db.json"
JSON-serverin käynnistys npm run server
Production build npm run build

Backend

Noden alustus npm init
kenttään scripts lisäys "start": "node index.js"
Suorita index.js npm start
Express asennus npm install express
Nodemon asennus npm install --save-dev nodemon
Käynnistä palvelin npm run dev
CORS (Cross-Origin Resource Sharing) asennus npm install cors
